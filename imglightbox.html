imgLightbox

Just another simple image lightbox.

IE11, mobile browser and ES5 compatible.

API

Include

Insert the .js and .css file in the page header:

<link href="imglightbox.min.css" rel="stylesheet" type="text/css" />
<script src="imglightbox.min.js"></script>

Insert this code in your html page:

<div id="imgLightbox" onclick="imgLightbox.close();">
  <div id="imgLightbox-button-area">
    <a title="open externally and close" id="imgLightbox-button-external">
      <span class="imgLightbox-button"><i class="fas fa-expand-alt"></i></span>
    </a>
    <a title="download and close" id="imgLightbox-button-download">
      <span class="imgLightbox-button"><i class="fas fa-arrow-alt-circle-down"></i></span>
    </a>
    <span title="close" class="imgLightbox-button" onclick="imgLightbox.close();"><i class="fas fa-times-circle"></i></span>
  </div>

  <div id="imgLightbox-image-area"></div>
</div>

Insert the references to FontAwesome for the lightbox icons:

<link href="fontawesome/fontawesome.min.css" rel="stylesheet" type="text/css" />
<link href="fontawesome/solid.min.css" rel="stylesheet" type="text/css" />

If you don't want to use FontAwesome, use this HTML instead and don't insert the references to FontAwesome:

<div id="imgLightbox" onclick="imgLightbox.close();">
  <div id="imgLightbox-button-area">
    <a title="open externally and close" id="imgLightbox-button-external">
      <span class="imgLightbox-button">&nearr;</i></span>
    </a>
    <a title="download and close" id="imgLightbox-button-download">
      <span class="imgLightbox-button">&DownArrowBar;</i></span>
    </a>
    <span title="close" class="imgLightbox-button" onclick="imgLightbox.close();">&Cross;</i></span>
  </div>

  <div id="imgLightbox-image-area"></div>
</div>

Get the library version:

alert( imgLightbox.config.version );

Set config.download (default is false):

imgLightbox.config.download = true;
imgLightbox.config.download = false;

Set config.externalOpen (default is false):

imgLightbox.config.externalOpen = true;
imgLightbox.config.externalOpen = false;

imgLightbox.open() method parameters:

imgLightbox.open(<filename: string>, <config: object>);

Samples

Open with the download button

imgLightbox.open('test1.jpg', {download: true});

Open with the download button and the external open button

imgLightbox.open('test1.jpg', {download: true, externalOpen: true});

Open with the config.download setting:

imgLightbox.open('test1.jpg');

Open with the external open button but no download button:

imgLightbox.open('test1.jpg', {externalOpen: true});

Small size image with the download button:

imgLightbox.open('test2.jpg', {download: true});

Small size image with the external open button:

imgLightbox.open('test2.jpg', {externalOpen: true});

Small size image with the config.download setting:

imgLightbox.open('test2.jpg');

Small size image without the download button:

imgLightbox.open('test2.jpg', {download: false});
